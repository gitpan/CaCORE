DatabaseCrossReference

CaCORE::Common::DatabaseCrossReference - Perl extension for DatabaseCrossReference.

ABSTRACT

The CaCORE::Common::DatabaseCrossReference is a Perl object representation of the CaCORE DatabaseCrossReference object.

SYNOPSIS

See the CaCORE::ApplicationService manpage.

DESCRIPTION

ATTRIBUTES of DatabaseCrossReference

The following are all the attributes of the DatabaseCrossReference object and their data types:

crossReferenceId

data type: string

dataSourceName

data type: string

id

data type: long

sourceType

data type: string

summary

data type: string

type

data type: string

  Note: Although you can also use the corresponding setter methods to set the
  attribute values, it is not recommended to do so unless you absolutely have
  to change the object's attributes.

ASSOCIATIONS of DatabaseCrossReference

The following are all the objects that are associated with the DatabaseCrossReference:

Collection of SNP:

Many to one assoication, use getSNP to get the associated SNP.

Collection of EngineeredGene:

Many to one assoication, use getEngineeredGene to get the associated EngineeredGene.

Collection of Gene:

Many to one assoication, use getGene to get the associated Gene.

Collection of NucleicAcidSequence:

Many to one assoication, use getNucleicAcidSequence to get the associated NucleicAcidSequence.
