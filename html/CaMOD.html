AbstractCancerModel

CaCORE::CaMOD::AbstractCancerModel - Perl extension for AbstractCancerModel.

ABSTRACT

The CaCORE::CaMOD::AbstractCancerModel is a Perl object representation of the CaCORE AbstractCancerModel object.

SYNOPSIS

See the CaCORE::ApplicationService manpage.

DESCRIPTION

ATTRIBUTES of AbstractCancerModel

The following are all the attributes of the AbstractCancerModel object and their data types:

experimentDesign

data type: string

id

data type: long

modelDescriptor

data type: string

  Note: Although you can also use the corresponding setter methods to set the
  attribute values, it is not recommended to do so unless you absolutely have
  to change the object's attributes.

ASSOCIATIONS of AbstractCancerModel

The following are all the objects that are associated with the AbstractCancerModel:

Collection of ApprovalStatus:

Many to one assoication, use getApprovalStatus to get the associated ApprovalStatus.

Collection of Availability:

Many to one assoication, use getAvailability to get the associated Availability.

Instance of Publication:

One to many assoication, use getPublicationCollection to get a collection of associated Publication.

SUPPORT

Please do not contact author directly. Send email to ncicb@pop.nci.nih.gov to request support or report a bug.
